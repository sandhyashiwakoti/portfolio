# Sandhya Shiwakoti — Portfolio

A dark-themed React portfolio built with Vite and Framer Motion.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run locally:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Deploy to Netlify

1. Push this folder to a GitHub repo
2. Go to netlify.com → "Add new site" → "Import from Git"
3. Set build command: `npm run build`
4. Set publish directory: `dist`
5. Click Deploy!

## Customizing

- **Projects & Skills**: Edit `src/data.js`
- **About text**: Edit `src/components/About.jsx`
- **Colors**: Edit CSS variables at the top of `src/index.css`
- **Resume link**: Add your CV PDF to the `public/` folder as `Sandhya_Shiwakoti_CV.pdf`
- **Portfolio URL**: Once deployed, update the portfolio link in your CV

## Tech Stack

- React 18
- Vite
- Framer Motion (animations)
- Lucide React (icons)
- Google Fonts: Syne + DM Sans
